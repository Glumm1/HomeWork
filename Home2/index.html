Робота в консолі

Команда Пояснення
cd (change directory) Команда використовується для навігації в структурі дерева каталогів.
mkdir (make directory) Команда використовується для створення нових директорій у файловій системі.
cp Команда, яка дозволяє копіювати файли або каталоги у файловій системі.
touch Команда, яка дозволяє створювати нові порожні файли.
hictory Команда, яка виводить на екран всі дії виконані в консолі
